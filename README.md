This project is built as part of a software engineering course at KTH (DD1367) in collaboration with and based on a project proposal by Tom Tit Experiment.

Our project is to create an interactive experiment or experience that is connected to the scientific principles of sound and/or air. The idea we have is a simulation-based experience where users place virtual objects on a screen to observe the movement and properties of sound in real time. The simulation realistically demonstrates the movement and properties of sound waves, such as interference and reflection, as they move across the screen and collide and interact with each other.

## Running

### Prebuilt

If on windows you can download the latest release from github. Just run the file and experience the project.

### From source

#### Prerequisites

- Godot Engine 4.x
- Git

#### Steps

1. Clone the repository

```bash
git clone https://github.com/PEOL0/DD1367-Interactive-Sound-Wave-Simulation.git
```

2. Open Godot
3. Import the project using the `project.godot` file
4. Press F5 to run the project
5. In the new window:
   - Place objects on the screen
   - Observe sound wave movement
   - Experiment with wave interference and reflection

### Building from source

To get more info on how to build the project from source please refer to the Godot documentation at <https://docs.godotengine.org/en/stable/tutorials/export/exporting_projects.html>. Note that you need to do step 1-3 above as well.
